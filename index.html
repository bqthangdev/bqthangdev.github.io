Image to Base64

Upload one or more images to convert them to Base64.

💡 Hướng dẫn sử dụng
  • Nhấn vào vùng upload hoặc kéo thả ảnh vào (PNG, JPG, GIF, WebP, SVG…). Có thể chọn nhiều ảnh cùng lúc.
  • Mỗi ảnh hiển thị kết quả riêng; chọn định dạng Base64 (chuỗi thuần) hoặc Data URL (có prefix data:image/…).
  • Nhấn Copy để sao chép hoặc Download để tải về file .txt.

Text Compare

Paste text in both panels and click Compare to see line-by-line differences.

💡 Hướng dẫn sử dụng
  • Dán nội dung vào ô Text A và Text B, sau đó nhấn Compare!.
  • Kết quả hiển thị theo dòng: nền xanh = thêm mới, nền đỏ = đã xóa, nền vàng = có thay đổi.
  • Trong các dòng thay đổi, các ký tự khác nhau được highlight thêm ở mức ký tự.
  • Nhấn Clear all để xóa toàn bộ và làm mới.

Markdown Reader

Upload a .md file or paste Markdown content to preview it.

💡 Hướng dẫn sử dụng
  • Nhấn Load .md file để tải file Markdown từ máy, hoặc paste trực tiếp vào ô bên trái.
  • Preview tự động cập nhật theo thời gian thực khi bạn nhập.
  • Chọn Side by side để xem song song hoặc Below để xem bên dưới.
  • Nhấn Clear để xóa nội dung và reset preview.
